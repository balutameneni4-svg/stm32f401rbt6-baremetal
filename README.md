

# STM32F401RBT6 Bare-Metal Programming

## Description
This repository contains bare-metal examples for STM32F401RBT6
using register-level programming (no HAL).

## Topics Covered
- GPIO
- ADC
- UART
- Interrupts

## Tools Used
- STM32F401RBT6
- ARM Cortex-M4
- Keil uVision 
- Logic Analyzer (Optional)

## How to Use
1. Clone the repository
2. Open project in Keil
3. Build and flash
